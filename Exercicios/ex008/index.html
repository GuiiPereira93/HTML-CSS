Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em Negrito usando a tag B, tag apenas de formatação, não semantica

Já nesta frase, temos um termo em negrito utilizando a tag STRONG, semantica (com sentido)

italico / Enfase

Neste exemplo, criamos um termo em italico utilizando a tag I(apenas estilo)

Já neste exemplo, criamos um termo com enfase, utilizando a tag ENFASE (semantica)

Textos Marcados

Podemos criar também um texto marcado utilizando a tag MARK. Entretanto, se eu quiser utilizar uma cor diferente eu posso utilizar a marcação STYLE e selecionar o subgrupo background para alterar a cor, que ficará somente neste paragrafo.

Para alterar em todo o texto e também em futuras marcações é necessário alterar na cabeça no site (head) assim ele herdará para os próximos marks, como este por exemplo, que ficou azul, como o paragrafo anterior está marcado dentro da tag, ele não irá alterar.

Já este novo paragrafo sim, irá ficar inteiro azul.

Textos grandes e pequenos

Estamos criando um texto grande utilizando a tag BIG, que já está descontinuada pel html5, e um texto pequeno utilizando a tag SMALL, que continua valida

Texto Deletado

Podemos marcar um texto como excluído utilizando a tag DEL

Texto Inserido

Podemos também dar enfase a um texto incluido posteriormente utilizando a tag INS

A tag INS irá sublinhar o texto, que também poderia ser feito com a tag U, como neste caso, mas a TAG U não tem semantica, apenas estilo ou seja, deveria ser feito via CSS

Texto Sobrescrito

Textos sobrescritos são os textos que tem caracteteres elevados, como por exemplo X20*2, onde utilizado a tag SUP.

Texto Subscrito

Já os textos subscritos, são textos que contem caracteres diminuídos como formulas quimicas, exemplo o H2O, que utilizamos a tag sub.
